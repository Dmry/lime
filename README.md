# lime
Tool for the application of Likhtman&amp; McLeish' reptation theory under active development.


## Requirements

* Boost >= 1.66
* Intel-tbb (optional for multithreading)
* C++17 (GCC => 7)
* GNU Scientific Library >= 2.4
